
# 盘中实时监控系统 - 操作指南

## 1. 竞价数据导入 (9:25后)

为了获取每个标的的**竞价成交金额**和**竞价涨幅**，本系统支持从同花顺导出数据并自动解析。

### 操作步骤:
1.  **打开同花顺**，进入自选股或相关板块界面。
2.  确保表格中包含以下列（列名不需要完全一致，但必须包含关键字）：
    *   **代码** (必选)
    *   **名称** (可选)
    *   **竞价金额** (或 `竞价额`, `早盘竞价金额`)
    *   **竞价涨幅** (或 `竞价涨幅%`)
3.  全选数据 (`Ctrl+A`)，复制 (`Ctrl+C`)。
4.  在 `data/input/call_auction/` 目录下创建一个新的 `.txt` 文件，文件名建议包含日期，例如 `Table_20260113.txt`。
5.  将复制的内容粘贴到该 TXT 文件中，**保存并关闭**。
    *   *注意：直接粘贴即可，系统会自动处理 Tab 和空格混合的情况。*

### 常见问题:
*   **编码问题**: 系统会自动尝试 `UTF-8` (默认) 和 `GBK` 编码，您无需手动转换。
*   **列对齐**: 系统采用智能行解析，即使数据看起来没对齐，也能正确读取。

## 2. 运行监控

在终端中执行：
```bash
python src/monitors/call_auction_screener.py
```

系统会自动加载最新的竞价文件，并结合实时行情进行筛选。
