# F佬/A大 核心操盘模式 (进阶优化版)

⚠️ **核心综述**: 本模式融合了 **A大断板反包** (超短博弈) 与 **F佬趋势抱团** (容量逻辑)，并结合了 **155监管** 下的实战心法。

---

## 1. 核心战法一：断板反包 (博弈弱转强)
> **"高辨识度人气股的超短线博弈"**

### 📌 选股标准 (T-1日 / 昨)
*   **核心身份**: 必须是**主线题材的核心容量票** (成交额大、非杂毛)。只有核心票才有资金愿意做反包。
*   **形态**: 昨日涨停，今日**爆量断板** (炸板或未能连板)，但**未跌破5日线**。
*   **情绪**: 必须是**分歧日**。大家都恐慌时，真正的龙头才会显现出承接力。

### ⚔️ 买入操作 (T日 / 今)
*   **✅ 深水低吸 (核心手法)**:
    *   在T日早盘急杀至**深水区 (-5%以下)** 或 **明显缩量回调** 时介入。
    *   **博弈逻辑**: 博弈日内修复，目标是收盘拉红甚至反包涨停 (大长腿)。
*   **❌ 拒绝追高**:
    *   除非是确认的主升浪加速，否则竞价大幅高开往往是兑现点，容易吃面。

### 📈 验证与卖出
*   **收红确认**: 无论T日盘中怎么走，**收盘必须收红** (符合A大焚诀核心)。
*   **卖出纪律**:
    *   **不反包即走**: T日低吸后，若T+1日不能强势反包或弱转强，坚决离场。
    *   **不做随手单**: 严禁盘中临时起意买入非计划内的票。

## 2. 核心战法二：🔥 焚诀 (趋势抱团)
> **"不是只有涨停才是涨，趋势才是大妖股的温床"**

### � 形态定义 (焚诀之相)
*   **连续放量阳线**: 股价沿着 **5日线** 攀升，连续走出温和放量的阳线。
*   **假阴真阳 (洗盘)**: 
    *   形态: 高开低走收绿(阴线)，但股价(收盘价)较昨日是**涨的**。
    *   逻辑: F佬认为只要不破趋势，这种K线是主力的洗盘动作，**不可轻易下车**。
*   **拒绝回调**: 分歧日不深跌，收盘价重心不断上移。

### ⚔️ 操作手法
*   **滚动做T**: 底仓不动，沿着5日线每天做T (急杀低吸，冲高高抛)。
*   **锁仓逻辑**: 只要维持 "放量阳线" 或 "缩量回调不破5日线"，就一直持有。
*   **离场信号**:
    *   **高位加速缩量**: 往往是见顶信号。
    *   **爆量长阴**: 确认退潮，清仓。

## 3. 辅助判断：155方针与异动
*   **控异动 (良性回调)**: 
    *   主力为了规避 "10天100%" 严重异动监管，会有意压盘。
    *   此时出现的断板或回调是**良性的**，是绝佳买点。
*   **渡劫 (空间打开)**:
    *   当股票即将触发严重异动时，会剧烈震荡。
    *   若能抗住分歧 (渡劫成功)，后市空间将彻底打开。
